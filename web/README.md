# StreetLore marketing site

Static landing page. No build step.

## Deploy on Netlify
- Drag this `web/` folder into Netlify, **or**
- Connect the repo and set **Publish directory** to `web` (leave build command empty).

Files: `index.html` (self-contained), `favicon.png`, `og-image.png`.
